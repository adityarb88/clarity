1. Datepicker - Auto Host Wrapped

a. ngModel

[(ngModel)]
{{model}} 
    
        <input clrDatepicker [(ngModel)]="model">
b. clrDatepicker
We do not support an Input for [clrDatepicker] since the user can use ngModel directly. The output is supported because we emit a Javascript Date object which <input type="text"> does not do.

(clrDatepickerChange)
{{date}} 
    
         <input clrDatepicker (clrDatepickerChange)="datepickerChange($event)">
